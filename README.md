## Farm graph
[ducklock.duckdns.org](ducklock.duckdns.org)

You can embed this chart into your website with an iframe.

```
<iframe 
    src="https://ducklock.duckdns.org/"
    width="600" 
    height="600" 
    frameborder="2" 
    allowfullscreen 
    title="Embedded Content">
</iframe>
```
